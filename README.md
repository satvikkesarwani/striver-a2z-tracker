# ⚡ A2Z DSA Tracker

Striver's A2Z DSA sheet, rebuilt with a clean dark UI — every question links **directly to LeetCode (preferred) or GeeksforGeeks**, never the tuf console.

## Features

- **474 problems · 18 topics**, same structure as the official A2Z sheet
- Click a problem → opens it on **LeetCode** (292), **GFG** (143), or Coding Ninjas (3); pure-theory items (36) open the free takeuforward article
- **Topic-wise progress bars** with per-topic Easy/Medium/Hard breakdown
- **Global Easy / Medium / Hard progress bars**
- Search, difficulty filter, hide-solved toggle
- Video-solution ▶ and editorial 📄 quick links per problem
- Progress saved in your browser (localStorage) + export/import as JSON
- Dark theme, responsive, zero dependencies

## Run locally

```bash
npm start        # serves on http://localhost:3000
```

Or just open `index.html` in a browser — it's fully static.
